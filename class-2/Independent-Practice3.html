I found out how to write emojis in HTML today😀 from here. I added the meta charset="UTF-8" element in my head, which defines the character set.

NB: The characters A, B, and C, are displayed by the numbers 65, 66, and 67. To let the browser understand that you are displaying a character, you must start the entity number with &# and end it with ; (semicolon).
