2. h tags

These are for adding headings in a web page, the headings can vary in size depending on what number you put after the 'h'. These tags run from h1 to h6, where the latter is the smallest heading.

The tiniest heading

3. strong tag

The 'strong' tag is to make things bold. I also found out that the 'b' tag does the same thing also.

4. em tag

The 'em' tag is to make words italic. You can also combine them: to bolden and italize the same word.

5. marquee tag

The marquee tag is used for scrolling piece of text or image displayed either horizontally across or vertically down your web site page depending on the settings.text My text is scrolling horizontally.This is super COOL!

6. mark tag

The 'mark' tag is used to highlight text that is of special interest or relevance in an HTML document. It marks the text with a yellow colour(I wonder if the colour can be customized?!)

7. sub tag

The 'sub' tag defines subscript text. Subscript text appears half a character below the normal line, and is sometimes rendered in a smaller font. Subscript text can be used for chemical formulas, like H2O.

8. sup tag

Use the 'sup' tag to define superscripted text, for eaxample: 23 = 8.
